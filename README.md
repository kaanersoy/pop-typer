# PopTyper ⌨️

App for impove your typing skills.

### 🔴 Demo 
You can visit the demo from: [pop-typer](https://pop-typer.vercel.app/)

## 📦 Installation

### 📰 Clone the project

```bash
git clone https://github.com/kaanersoy/pop-typer.git
```

### 🔻 Install Dependencies

With yarn:
```bash
cd ./pop-typer && yarn
```

### 🏃‍♂️ Start the project

With yarn:
```bash
yarn dev
```
